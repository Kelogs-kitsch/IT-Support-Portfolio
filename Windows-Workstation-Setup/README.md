# Windows Workstation Setup Guide
This documentation provides the complete procedure for configuring a new employee workstation, including account setup, software installation, system configuration, and security settings.

## System Information
Verified system hardware and OS build using msinfo32. 
Documented CPU, RAM, device model, and Windows version.

Press Windows + R

Type: msinfo32

Press Enter

Created a standard user account for the new employee using Windows account settings. 
Ensured account type remains Standard to maintain security.

Install all available updates

## Storage Configuration and Cleanup
Performed disk cleanup to remove temporary and unused system files. 
Confirmed Storage Sense settings for automatic cleanup.

Press the Windows key + R to open the Run dialog box.

Type cleanmgr in the box.

Press Enter or click OK.

Select the drive you want to clean and click OK
