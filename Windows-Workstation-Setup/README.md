# Windows Workstation Setup Guide
This documentation provides the complete procedure for configuring a new employee workstation, including account setup, software installation, system configuration, and security settings.

## System Information
Verified system hardware and OS build using msinfo32. 
Documented CPU, RAM, device model, and Windows version.

Press Windows + R

Type: msinfo32

Press Enter
